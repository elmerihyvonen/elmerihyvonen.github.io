CAPSTONE PROJECT - 3D KARTAN LUOMINEN JA VISUALISOINTI SELAIMELLA

Projekti on tehty Turun yliopiston Capstone kurssille ryhmätyönä. Toteutimme kuuden henkilön projektiryhmän kanssa 3D kartan Lidar tutkan kanssa. Keräsimme tutkalla 3D dataa Turun yliopiston kampusalueelta. Data prosessoitiin Googlen Cartographer työkaluilla. Cartographer mahdollistaa simultaneous localization and mapping (SLAM) -toteutuksen sekä 2D että 3D kartoille. Kartan visualisoimisessa web selaimen kautta hyödynsimme pisteavaruuksien visualisointiin tarkoitettua web vieweriä. Ratkaisu thetiin Google Cloudiin. Minun vastuualueeni projektissa oli tekninen toteutus.


Pääset lukemaan yliopiston sivuilla julkaistun projektin esittelyn tästä.

Työssä käytetyt tekniikat & työkalut & alustat & yms :

  • Linux
  • Robot Operating System (ROS)
  • Google Cartographer SLAM
  • Google Cloud
  • LiDAR laser scanner
  • Point cloud web viewer
